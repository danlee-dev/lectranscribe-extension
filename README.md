# LecTranscribe Chrome Extension

LMS 강의 및 YouTube 영상을 자동 감지하여 전사하는 Chrome 확장 프로그램입니다.

---

## 설치 방법

### 1. 파일 다운로드

- 이 페이지 오른쪽 위의 초록색 **Code** 버튼 클릭
- **Download ZIP** 클릭
- 다운로드된 ZIP 파일 압축 해제

### 2. Chrome 확장 프로그램 페이지 열기

- Chrome 주소창에 `chrome://extensions` 입력
- 오른쪽 상단의 **개발자 모드** 토글을 켜기

### 3. 확장 프로그램 로드

- 왼쪽 위의 **압축해제된 확장 프로그램을 로드합니다** 클릭
- 압축 해제한 폴더 안의 `lectranscribe-extension-main` 폴더를 선택

---

## Installation (English)

### 1. Download files

- Click the green **Code** button at the top right of this page
- Click **Download ZIP**
- Extract the downloaded ZIP file

### 2. Open Chrome extensions page

- Enter `chrome://extensions` in Chrome's address bar
- Enable **Developer mode** toggle in the top right

### 3. Load the extension

- Click **Load unpacked** in the top left
- Select the `lectranscribe-extension-main` folder inside the extracted folder

---

## 사용 방법

### LMS 강의 전사
1. 학교 LMS(고려대 블랙보드 등)에서 강의 영상 재생
2. 영상이 감지되면 화면 오른쪽 아래에 LecTranscribe 버튼이 나타남
3. 버튼 클릭 시 자동으로 전사 시작

### YouTube 영상 전사
1. YouTube에서 전사할 영상 열기
2. 화면 오른쪽 아래에 LecTranscribe 버튼이 나타남
3. 버튼 클릭 시 자동으로 전사 시작
